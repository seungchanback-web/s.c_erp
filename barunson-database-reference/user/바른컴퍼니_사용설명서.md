# 바른컴퍼니 발주운영 시스템 사용설명서

> 접속: http://localhost:12026

## 1. 대시보드

사이트 접속 시 첫 화면. 발주 현황을 한눈에 파악할 수 있습니다.

### KPI 카드 (상단 4개)
| 카드 | 설명 |
|------|------|
| **긴급재고** | 가용재고가 마이너스인 품목 수 (클릭 시 재고현황 이동) |
| **위험재고** | 가용재고 ≤ 월사용량인 품목 수 |
| **이번달 발주건수** | 당월 생성된 PO 수 |
| **미수령 발주** | 수령 완료되지 않은 PO 수 |

### 발주 파이프라인
발주 상태별 흐름을 시각화합니다.
- **대기** → **발송** → **확인** → **수령중** → **완료** | **취소**
- 각 단계의 건수와 수량이 표시됩니다
- 클릭하면 발주현황 페이지로 이동합니다

### 전체 입고율 (도넛 게이지)
- 전체 발주수량 대비 입고된 수량의 비율(%)
- 0%에 가까우면 입고가 지연되고 있다는 의미입니다

### 거래처별 발주 비중
- 상위 5개 거래처의 발주 건수/수량을 수평 바 차트로 표시
- 특정 거래처에 발주가 편중되어 있는지 확인 가능

### 취소율 경고 배너
- 취소율 20% 이상: 주황 경고
- 취소율 30% 이상: 빨강 경고
- 취소가 빈번하면 발주 프로세스 점검이 필요합니다

### 최근 발주
- 최근 5건의 PO 번호, 거래처, 상태, 수량 표시
- PO 번호 클릭 시 발주현황으로 이동

### 원재료 매입현황
- 연간 거래처별/품목별 매입 수량 및 금액
- 월별 추이를 테이블로 확인

## 2. 재고현황

ERP 스마트재고현황 데이터를 기반으로 품목별 재고 상태를 분류합니다.

### 데이터 로드 방법
- **자동**: 서버에 저장된 최신 엑셀 데이터 사용
- **수동**: 엑셀 파일(.xls/.xlsx)을 드래그&드롭하여 갱신
- **데이터 갱신** 버튼: 서버에서 최신 데이터 재로드

### 분류 기준
| 분류 | 조건 | 색상 |
|------|------|------|
| **긴급** | 가용재고 < 0 (마이너스) | 빨강 |
| **위험** | 가용재고 ≤ 월사용량 | 주황 |
| **안전** | 가용재고 > 월사용량 | 초록 |
| **매출없음** | 최근 3개월 매출 0 | 회색 |

### 테이블 컬럼 설명
| 컬럼 | 설명 |
|------|------|
| 분류 | 긴급/위험/안전 뱃지 |
| 품목코드 | 제품 SKU |
| 특이사항 | 소진 후 단종/보류/신규/발주완료 등 뱃지 (클릭하여 편집) |
| 브랜드 | 제품 브랜드 |
| 카드구분 | 내지/청첩장/부속 등 |
| 현재고 | ERP 현재고 수량 |
| 요청량 | 출고 요청 중인 수량 |
| 가용재고 | 현재고 - 요청량 (실제 사용 가능한 재고) |
| 3개월매출 | 최근 3개월 출고 합계 |
| 월사용량 | 3개월매출 ÷ 3 |
| 잔여월 | 가용재고 ÷ 월사용량 (재고로 버틸 수 있는 개월 수) |
| 부족수량 | 발주가 필요한 수량 |
| 발주수량 | 직접 입력하는 발주 수량 (R 단위 자동 변환) |
| 선택(✓) | 발주 대상 체크박스 |

### 발주 워크플로우
1. 긴급/위험 탭에서 발주 필요 품목 확인
2. 발주수량 입력 (자동입력 버튼으로 일괄 채움 가능)
3. **미리보기** 버튼으로 발주서 확인
4. **선택완료 → 발주생성** 클릭하여 발주생성 페이지로 이동

## 3. 필수 자동발주

특정 품목의 재고가 설정값 이하로 떨어지면 자동으로 PO를 생성하는 기능입니다.

### 설정 항목
| 항목 | 설명 |
|------|------|
| **품목코드** | 자동발주 대상 품목 |
| **거래처** | 발주할 거래처 선택 |
| **최소재고** | 가용재고가 이 값 이하면 발주 트리거 |
| **발주수량** | 자동발주 시 주문할 고정 수량 |
| **활성** | ON/OFF로 개별 품목 활성화/비활성화 |

### 사용법
1. 품목별 최소재고와 발주수량을 설정 (인라인 수정 가능)
2. 거래처를 선택
3. **자동발주 실행** 버튼 클릭
4. 가용재고 ≤ 최소재고인 품목에 대해 PO가 자동 생성됨
5. 같은 날 이미 자동발주된 품목은 중복 생성되지 않음
6. 생성된 PO는 발주현황에서 "자동발주" 타입으로 확인 가능

### 품목 추가/삭제
- **+ 품목추가** 버튼으로 새 품목 등록
- **×** 버튼으로 목록에서 삭제

## 4. 발주생성

재고현황에서 선택한 품목들을 거래처별로 분리하여 발주서를 생성합니다.

### 탭 구분
- **원재료**: 제지사(용지 공급업체) 기준으로 그룹핑
- **후공정**: 공정별 업체 기준으로 그룹핑 (한 품목이 여러 업체에 나올 수 있음)

### 발주서 생성 과정
1. 거래처별 카드에서 품목 목록 확인
2. 납기일, 메모 입력 (선택사항)
3. 거래처별 **발주저장** 버튼 클릭
4. PO가 "대기" 상태로 생성됨

---

## 5. 발주현황

생성된 모든 PO를 관리합니다.

### 상태 필터
전체 / 대기 / 발송 / 확인 / 수령중 / 완료 / 취소

### PO 상태 흐름
```
대기 → 발송 → 확인 → 수령시작 → 완료
              ↘ 취소
```

### 각 상태별 가능한 액션
| 상태 | 가능한 액션 |
|------|------------|
| **대기** | 발송, 취소 |
| **발송** | (거래처가 확인하면 "확인"으로 변경) |
| **확인** | 수령시작 |
| **수령중** | (수령관리에서 수령 등록) |
| **완료** | (모든 품목 수령 완료 시 자동) |

- **발송** 시 Google Sheets에 발주 데이터가 자동 동기화됩니다
- **취소** 시 Google Sheets에서 취소선+빨간글씨 처리됩니다

---

## 6. 수령관리

발주한 물품의 입고(수령)를 등록합니다.

### 수령 등록 과정
1. PO 선택 (완료/취소 제외)
2. 품목별 수령수량 입력
3. 수령일, 수령자, 메모 입력
4. **수령 저장** 클릭
5. 전체 수령 시 PO 상태가 자동으로 "완료"로 변경

## 7. 거래명세

거래처별 거래명세서를 생성하고 관리합니다.

- 거래처, 기간, 품목 선택하여 명세서 생성
- PDF 출력 가능
- 첨부파일 업로드 지원

---

## 8. 거래처

등록된 거래처 정보를 관리합니다.

### 분류
- **원재료**: 용지/원자재 공급업체 (대한통상, 삼원제지, 두성종이, 한솔PNS, 주식회사 서경)
- **후공정**: 가공/인쇄 업체 (코리아패키지, 예지가)

### 거래처 카드 정보
- 거래처코드, 담당자, 전화번호, 이메일, 카카오톡, 메모

---

## 9. 미팅일지

거래처 미팅 기록을 관리합니다.
- 제목, 거래처, 날짜, 내용 기록
- 거래처별 필터링 가능

## 10. MRP

자재소요계획(Material Requirements Planning) 기능입니다.
- BOM(자재명세서) 등록
- 월별 생산계획 수립
- MRP 실행으로 필요 원자재 자동 산출

## 11. 설정

### 발주 기준 설정
| 설정 | 기본값 | 설명 |
|------|--------|------|
| 매출 기간 | 3개월 | 월사용량 계산 기준 기간 |
| 발주 배수 | 1개월 | 몇 개월치를 발주할지 |
| 올림 단위 | 1,000 | 발주수량 올림 단위 |
| 위험 배수 | 1.0 | 위험재고 판단 기준 (월사용량 × 배수) |

### 월별 가중치 (계절성 지수)
- 2024~2025년 전체 품목의 월별 출고량 기반
- 전월 대비 증감률을 보여줌
- 참고용 거시 트렌드 (개별 품목 적용에는 한계)

## 용어 정리

| 용어 | 설명 |
|------|------|
| **PO** | Purchase Order (발주서) |
| **가용재고** | 현재고 - 요청량 (실제 사용 가능한 수량) |
| **월사용량** | 최근 3개월 출고량 ÷ 3 |
| **잔여월** | 가용재고 ÷ 월사용량 |
| **R (연)** | 발주 단위 = 수량 ÷ 500 ÷ 절 ÷ 조판 |
| **부족수량** | 추가 발주가 필요한 수량 |
