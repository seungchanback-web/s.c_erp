자동발주 계산 기준
리드타임(LT) 기반 전략발주:
  LT: 품목별 설정값 > 생산지 기본값 (중국 50일, 한국 7일, 더기프트 14일)
  목표재고 = (일평균출고 × LT) + 안전재고
  안전재고: 중국 = 월출고 × 1.5, 한국/기타 = 월출고 × 0.5
  부족수량 = 목표재고 - 가용재고 (음수면 0)
긴급(잔여 ≤ 14일): 거래처 주간한도 무시, 즉시 발주
위험(잔여 ≤ 21일): 거래처당 주 6건 제한 적용
스킵: 월출고 0 / 미완료 PO / 가용재고 ≥ 목표재고
추천수량 (단가 구간 최적화):
  부족수량 이상 & 최저 단가 구간 자동 선택
단가: 장당 위안화(¥) (중국 상품별 단가표 기준, 2025.5.1)
예상금액 = 추천수량 × 단가
